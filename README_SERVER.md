# 去笔迹 AI 服务 (Local Server)

这是一个基于 Python + OpenCV 的本地服务，用于移除试卷图片中的手写笔迹（主要是红色和蓝色墨水）。

## 1. 环境准备

确保你已经安装了 Python (建议 3.8+)。
项目依赖已经在 `server/requirements.txt` 中列出。

如果你需要手动安装依赖：
```bash
pip install -r server/requirements.txt
```

## 2. 启动服务

**方法 A：使用脚本（推荐）**
双击项目根目录下的 `start_server.bat` 脚本即可启动。

**方法 B：命令行启动**
在项目根目录下运行：
```bash
python server/main.py
```

服务启动后会监听 `http://0.0.0.0:8001`。

## 3. 小程序配置

在 `pages/add/add.js` 中，`SERVER_URL` 配置了服务器地址。
- 模拟器调试：默认使用 `http://127.0.0.1:8001/remove-handwriting`
- 真机调试：需要将 `127.0.0.1` 修改为你电脑的局域网 IP 地址（如 `192.168.1.5`），并确保手机和电脑在同一 WiFi 下。

## 4. 原理说明

该服务使用 OpenCV 库进行图像处理：
1. **颜色分割**：在 HSV 颜色空间中识别红色和蓝色区域（常见的批改颜色）。
2. **掩膜生成**：生成笔迹的 Mask，并进行膨胀操作以覆盖边缘。
3. **图像修复 (Inpainting)**：使用 Telea 算法根据周围像素修复 Mask 区域，还原试卷背景。

## 5. 注意事项

- 请确保微信开发者工具中勾选了 **"详情 -> 本地设置 -> 不校验合法域名..."** 选项。
- 目前的算法主要针对彩色笔迹（红/蓝），对于黑色笔迹效果可能有限（因为难以与黑色打印字体区分）。
